### Screen States During Workout

| State | Screen | Duration | User Actions |
|-------|--------|----------|--------------|
| **Pre-Start** | Workout Overview | Variable | Review exercises, tap "Start" |
| **Warm-Up** | Warm-up list | 5 min | Follow along, tap "Complete" |
| **Active Exercise** | Exercise tracking | ~8 min/exercise | Log sets, reps, weight, RPE |
| **Rest Period** | Rest timer | 60-120 sec | Wait, prepare for next set |
| **Between Exercises** | Exercise transition | 5-10 sec | Auto-advance or tap "Next" |
| **Safety Checkpoint** | Modal interrupt | 10 min | Take break, acknowledge |
| **Cool-Down** | Cool-down list | 5 min | Follow along, tap "Finish" |
| **Complete** | Summary screen | Variable | Review stats, add notes |

## State Management

### Workout State Machine

```javascript
const workoutStates = {
  NOT_STARTED: 'not_started',
  WARMUP: 'warmup',
  ACTIVE_EXERCISE: 'active_exercise',
  REST: 'rest',
  EXERCISE_COMPLETE: 'exercise_complete',
  SAFETY_CHECKPOINT: 'safety_checkpoint',
  COOLDOWN: 'cooldown',
  COMPLETE: 'complete'
};

const workoutState = {
  workout_id: 'uuid',
  status: 'in_progress',
  current_state: 'active_exercise',
  started_at: '2025-11-24T14:30:00Z',
  
  // Exercise tracking
  current_exercise_index: 0, // 0-5 (6 exercises)
  current_set: 1,
  total_sets_for_exercise: 4,
  
  // Set data for current exercise
  completed_sets: [
    { set: 1, reps: 10, weight: '30 lbs', rpe: 7, rest_seconds: 88 },
    { set: 2, reps: 10, weight: '30 lbs', rpe: 7, rest_seconds: 92 }
  ],
  
  // Timer
  workout_elapsed_seconds: 1044, // 17:24
  rest_timer_seconds: 90,
  rest_started_at: null,
  
  // Safety
  safety_checkpoints_triggered: [
    { type: 'binder_break', time: 2700, action: 'break_taken', duration: 600 }
  ],
  
  // Overall progress
  exercises_completed: 1,
  total_volume_lbs: 1140,
  warmup_completed: true,
  cooldown_completed: false
};
```

### Local Storage Strategy

**Persist state every time user logs a set:**
```javascript
// Save to device storage (React Native AsyncStorage)
await AsyncStorage.setItem(
  `workout_${workout_id}_state`,
  JSON.stringify(workoutState)
);

// Also sync to backend
await API.post(`/workouts/${workout_id}/sync-state`, workoutState);
```

**Recovery on app crash/close:**
```javascript
// On app reopen
const savedState = await AsyncStorage.getItem(`workout_${workout_id}_state`);

if (savedState && JSON.parse(savedState).status === 'in_progress') {
  // Show modal: "Resume workout from where you left off?"
  // If yes, restore state and navigate to correct screen
}
```
