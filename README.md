# TransFitness

Safety-first fitness app for trans and gender-diverse people.

## Quick Start

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Copy `.env.example` to `.env` and add your Supabase credentials
3. Start the app:
   ```bash
   pnpm start
   ```
